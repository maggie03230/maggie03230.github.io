How to use github pages

Github pages can be your free site. But, how? Here I don't use Hexo or Jeklly. But create a pure my project.

Step1:

Add repository and create your username.github.io.

Step2:

Add index.html.

Step3:

How to add other pages? Here I create a folder named "templates", add a page named "blog_1.html", add an link in index.html whose href = templates/blog_1.html. Then you can jump to see the blog_1 page.

Step4:

How to add css/js files? When I launched automatic page generator with a theme, a folder named "stylesheets" was created. And, there is a file named "stylesheet.css" in it. In index.html, add <link rel="stylesheet" type="text/css" href="stylesheets/stylesheet.css" media="screen">
